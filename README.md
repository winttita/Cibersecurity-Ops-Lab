# Portafolio de Operaciones de Ciberseguridad (SecOps)

¡Hola! Soy un Analista de Ciberseguridad en formación enfocado en **Defensa Activa (Blue Team)** y **Hacking Ético (Red Team)**.
Este repositorio documenta mi viaje práctico aprendiendo a desplegar, atacar y defender infraestructuras críticas.

## Metodología
Todo el contenido aquí sigue la filosofía **"Learning by Doing"**:
1.  **Despliegue:** Configuración de laboratorios y herramientas (Infrastructure as Code).
2.  **Ataque:** Simulación de amenazas reales (Kill Chain).
3.  **Defensa:** Detección y respuesta con SIEM/SOAR.
